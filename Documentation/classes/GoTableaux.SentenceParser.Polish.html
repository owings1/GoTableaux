Parses an atomic sentence from a string that starts with an atomic symbol.

parseAtomic(string $sentenceStr) : \GoTableaux\Sentence
Inherited

This is the default implementation for parsing an atomic sentence from a string that starts with an atomic symbol. This implementation expects the next symbol to be either a separator (space) character, or a subscript character followed by an integer. If a subscript is not given, it will be assigned 0.
