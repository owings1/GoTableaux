Implements the Reflexive Rule.



A node is reflexive exactly when both indexes are equal. This rule  searches for any indexes on the branch for which there is not currently  a reflexive node. If one is found, it adds a reflexive node with the  class AccessNode. If more than one are found, it adds  a node for least index.
