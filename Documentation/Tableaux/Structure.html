Class: Structure

Class Structure

Class Overview
Class Overview | Property Summary | Properties Detail | Method Summary | Methods Detail

Represents the proper 'tree' structure of a tableau.

This is a simple composition structure class which takes a Tableau object and creates a tree structure from its branches. Thinking of a branch as a tuple of nodes, a single node can appear on more than one branch. This is helpful in applying rules. This class collapses common nodes into one structure. A structure comprises an array of Node objects, and an array of self-similar Structure objects.
